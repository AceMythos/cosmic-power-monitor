# Cosmic Power Monitor — Project Report

## Overview
COSMIC desktop applet showing real-time battery power draw (watts) in the panel, equivalent to the GNOME extension "Power battery Consumption Watt Meter."

Built with Rust + libcosmic, data sourced from UPower via D-Bus (zbus v5).

## Architecture

```
main.rs              entry point, calls cosmic::applet::run::<PowerMonitor>
app.rs               Application impl: view, view_window, update, subscription, popup
battery.rs           UPower DBus poller → BatteryData
resources/*.desktop  desktop entry with X-CosmicApplet=true
```

### Data Flow
1. `battery.rs` polls `org.freedesktop.UPower.GetDisplayDevice` every 3s
2. Reads `EnergyRate`, `Percentage`, `State`, `TimeToEmpty`, `TimeToFull`, `Energy`, `EnergyFull`
3. App subscription receives `BatteryData` → updates panel icon + label
4. Panel shows `±X.X W` (red=discharging, green=charging) with battery icon
5. Popup shows percentage, status, watts, energy, time remaining

```sh

just install        # sudo install to /usr/local/bin + desktop entry

```
